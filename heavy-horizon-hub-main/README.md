# Heavy Horizon - Construction Equipment Rental & Sales

Heavy Horizon is a premier platform for renting and buying high-quality heavy construction equipment. We specialize in backhoe loaders, excavators, and other essential earthmoving machinery.

## Features

- **Equipment Rental**: Browse our available fleet for short-term and long-term rentals.
- **Sales**: Purchase certified used and new equipment.
- **Detailed specs**: View comprehensive specifications for all machinery.
- **Inquiry System**: Easy contact forms to request quotes or schedule viewings.

## Getting Started

To get started with development locally:

1. Clone the repository
   ```sh
   git clone <repository-url>
   ```

2. Install dependencies
   ```sh
   npm install
   ```

3. Start the development server
   ```sh
   npm run dev
   ```

## Technologies Used
